# Concordia  
**«Гармония через конфликт»**  
*(Ролевая игра с акцентом на экзистенциальный выбор и социальные последствия)*  

---

## Концепция  
**Игровой цикл строится на парадоксе:**  
_Чтобы исцелить себя, придётся погрузиться в эмоциональный хаос других._  
Вы — студент, обнаруживший, что подавленные эмоции материализуются в альтернативной реальности. Ваша задача не сражаться, а **переосмыслить** их через тактический карточный геймплей и моральные дилеммы.

---

## Сеттинг  
### Верхний мир: **Кампус**  
- Университет с аудиториями, библиотекой, спортзалом и общежитием.  
- **NPC с деструктивными паттернами (имена раскрываются через взаимодействие):**  
  - *Девушка с горящими глазами* (гнев) — фанатичная активистка, сжигающая себя изнутри.  
  - *Парень в лабораторном халате* (страх) — гений, запертый в подвале.  
  - *Мужчина с потёртым портфелем* (стыд) — бывший вундеркинд, подменяющий реальность лекциями.  

### Нижний мир: **Искажённые зоны**  
- Библиотека → **Ловушка перфекционизма**
- Общежитие → **Колония прокрастинации**

## Основные механики  
### 1. Карты-отношения  
- Каждая карта привязана к NPC и имеет **два режима:**  
  | **Светлая** (доверие)                                                             | **Теневая** (предательство)               |
  | --------------------------------------------------------------------------------- | ----------------------------------------- |
  | *Гнев активистки:* Урон + очистка дебаффов                                        | *Ярость фанатика:* Урон всем, включая вас |
  | *Страх затворника:* Противник пропускает ход + активация щита после 2 хода игрока | *Паранойя гения:* Игрок пропускает 1 ход  |

- **Прогресс отношений:**  
  - После 3 успешных взаимодействий NPC раскрывает имя (например, *Девушка с горящими глазами → Лина*).  
  - Имя открывает доступ к уникальным комбо-картам (например, *Лина + Парень в халате → «Недоверчивый союз»*).  

### 2. Динамическая сложность  
- **Адаптивный ИИ:**  
  - Если игрок использует 3+ атакующих карт → враги отращивают броню.  
  - Если фармит защиту → появляются **Эмоциональные вампиры**, сосущие ресурсы.  
- **Боссы-отражения:** Финальный противник каждого акта — гибрид ваших самых использованных карт.  

### 3. Прогрессия через потери  
- **Нет левел-апа** — вместо этого:  
  - Потеря карты → открывается слот для **руны подавления** (пассивные способности ценой части личности).  
  - "Смерть" NPC → его аудитория становится новой боевой ареной с уникальными дебафами.

## Техническая реализация  
### Ядро проекта  
- **Движок:** C++17 + SDL3 (окно/ввод) + Vulkan (рендер карт/эффектов).  
- **Генерация мира:** Комнаты собираются из модулей по принципу «эмоционального резонанса» (гнев → острые углы, страх → замкнутые пространства).

### Этапы разработки  
1. **Прототип за 2 недели:**  
   - 1 локация (Библиотека)  
   - 2 NPC с полным циклом мутации карт  
   - Система «эмоционального отклика» (NPC реагируют на ваш стиль боя)  

2. **Альфа за 2 месяца:**  
   - 5 боссов с алгоритмами кражи карт  
   - Система ветвящихся диалогов (влияет на элементы интерфейса)  

---

## Риски и решения  
| **Угроза**                   | **Страховка**                                                          |
| ---------------------------- | ---------------------------------------------------------------------- |
| Слишком сложная мутация карт | Ввести «якоря стабильности» — 3 базовые карты, которые нельзя потерять |
| Плохая читаемость интерфейса | Режим «Кризис» — автоматическое упрощение HUD при 20% HP               |
| Отсутствие катарсиса         | Финал-триггер: игрок сам становится боссом для нового игрока           |

---

## Уникальность проекта  
- **Социальный симулятор как тактическая карта:** Каждый диалог — расстановка фигур для будущего боя.  
- **Персонажи как многослойный ресурс:** Описания → имена → комбо-карты создают эффект постепенного погружения.  
- **Нет «игрового надзирателя»** — смерть NPC и разрушение карт не блокируют прогресс, но делают мир опаснее.
